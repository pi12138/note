# mysql 命令

- help command 查看命令帮助

## show命令

| show databases;              | 显示所有数据库                       |
| ---------------------------- | ------------------------------------ |
| show tables;                 | 显示所有表                           |
| show columns from table_name | 显示表的所有列（类似desc table_name) |
| show create database db_name | 显示创建数据库的sql语句              |
| show create table table_name | 显示创建表的sql语句                  |

## use命令

| use database; | 使用指定的数据库 |
| ------------- | ---------------- |
|               |                  |
